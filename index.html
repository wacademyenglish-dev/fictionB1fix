System Architecture

WHO

We use WHO to describe or explain a Person.

"Mio is a writer WHO loves science fiction."

"The man WHO has the sword is a knight."

System Architecture

WHICH

We use WHICH to describe an Object, Thing, or Animal.

"A hoverboard is a machine WHICH flies."

"She has a weapon WHICH shoots lasers."

*Note: You can also use "THAT" for things and people (e.g., A machine THAT flies).*

System Architecture

WHERE

We use WHERE to describe a Place or Location.

"A castle is a place WHERE the king lives."

"This is the planet WHERE they crashed."
